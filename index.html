Bab 2 - Apa Itu Website?

Bab 2

Apa Itu Website?

Memahami dasar sebelum mulai membuat website.

Pengertian Website

Website adalah kumpulan halaman yang dapat dibuka menggunakan browser seperti Google Chrome, Firefox, Microsoft Edge, atau Safari. Halaman-halaman tersebut berisi informasi, gambar, video, tombol, formulir, dan berbagai fitur lainnya.

Setiap website memiliki alamat unik yang disebut domain, misalnya:

  • www.google.com
  • www.wikipedia.org
  • www.github.com

Contoh Website

  • 🔍 Mesin pencari (contoh: Google)
  • 📺 Platform video
  • 🛒 Toko online
  • 📚 Website belajar
  • 📰 Portal berita
  • 🎮 Website game

Website Dibuat Menggunakan Apa?

Sebuah website modern biasanya dibuat menggunakan tiga teknologi utama:

  • HTML → Membuat struktur halaman.
  • CSS → Mengatur tampilan dan warna.
  • JavaScript → Menambahkan interaksi dan logika.

Ilustrasi Sederhana

HTML = Kerangka Rumah CSS = Cat & Dekorasi JavaScript = Listrik, Lampu, dan Pintu Otomatis

Bayangkan sebuah rumah. HTML adalah rangka dan dindingnya, CSS membuat rumah terlihat indah, sedangkan JavaScript membuat rumah menjadi interaktif, misalnya lampu yang bisa dinyalakan atau pintu yang terbuka otomatis.

Latihan

Coba jawab pertanyaan berikut:

  1. Apa yang dimaksud dengan website?
  2. Sebutkan tiga teknologi utama pembuat website.
  3. Apa fungsi HTML?
  4. Apa fungsi CSS?
  5. Apa fungsi JavaScript?

Bab 3 - Cara Kerja Internet

Bab 3

Cara Kerja Internet

Memahami bagaimana website sampai ke layar perangkatmu.

📘 Bab 3 dari 30 • ⏱ Waktu baca: ±6 menit

Apa itu Internet?

Internet adalah jaringan komputer terbesar di dunia yang menghubungkan miliaran perangkat agar dapat saling bertukar informasi.

Bagaimana Website Bisa Muncul?

Ketika kamu mengetik alamat website, prosesnya seperti ini:

Kamu ↓ Browser ↓ Internet ↓ Server ↓ Mengirim File HTML, CSS, JavaScript ↓ Browser Menampilkan Website

Penjelasan Setiap Bagian

Browser adalah aplikasi seperti Chrome atau Firefox yang membuka website.

Server adalah komputer yang menyimpan file website selama 24 jam.

HTML, CSS, dan JavaScript dikirim dari server ke browser agar halaman bisa ditampilkan.

Contoh Kehidupan Sehari-hari

Bayangkan kamu memesan makanan melalui aplikasi.
  • Kamu = Pengguna
  • Aplikasi = Browser
  • Restoran = Server
  • Makanan = File Website

Saat pesanan selesai dibuat, restoran mengirim makanan kepadamu. Begitu juga server mengirim file website ke browser.

Istilah Penting

  • 🌐 Internet → Jaringan global.
  • 💻 Browser → Pembuka website.
  • 🖥 Server → Penyimpan website.
  • 📄 HTML → Struktur website.
  • 🎨 CSS → Tampilan website.
  • ⚙ JavaScript → Interaksi website.

Latihan

  1. Apa fungsi browser?
  2. Apa fungsi server?
  3. Apa yang dikirim server ke browser?
  4. Sebutkan tiga contoh browser.

📖 Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML merupakan bahasa markup yang digunakan untuk membuat struktur sebuah halaman website.

HTML bukan bahasa pemrograman. HTML hanya bertugas mengatur isi halaman seperti judul, paragraf, gambar, tombol, tabel, formulir, dan masih banyak lagi.

📜 Sejarah HTML

HTML pertama kali diperkenalkan pada tahun 1991 oleh seorang ilmuwan komputer bernama Tim Berners-Lee. Tujuannya adalah agar para peneliti dapat saling berbagi dokumen melalui internet dengan mudah.

Seiring berkembangnya internet, HTML terus mengalami pembaruan hingga saat ini menggunakan standar HTML5.

🏠 Analogi HTML

Bayangkan kamu ingin membangun sebuah rumah.

  • 🧱 HTML = Kerangka rumah
  • 🎨 CSS = Cat dan dekorasi rumah
  • ⚡ JavaScript = Lampu, pintu otomatis, dan fitur elektronik

Tanpa HTML, rumah tidak memiliki bentuk. Tanpa CSS, rumah terlihat polos. Tanpa JavaScript, rumah tidak memiliki fitur interaktif.

💻 Website Pertama

<!DOCTYPE html>
<html>
<head>
<title>Website Pertama</title>
</head>

<body>

<h1>Halo Dunia!</h1>

<p>Ini adalah website pertamaku.</p>

</body>

</html>

🎯 Mengenal Heading

Heading adalah judul pada sebuah halaman website. Heading membantu pengunjung memahami isi halaman dan membuat tampilan lebih rapi.

HTML memiliki 6 tingkat heading, mulai dari <h1> hingga <h6>. 

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<h3>Judul Kecil</h3>

📝 Membuat Paragraf

Untuk menampilkan teks biasa, HTML menggunakan tag <p>. 

<p>
Halo, saya sedang belajar HTML.
</p>

Paragraf digunakan untuk menjelaskan informasi kepada pembaca.

💻 Menulis Kode HTML

Sekarang gabungkan heading dan paragraf menjadi satu halaman sederhana. 

<h1>Belajar HTML</h1>

<p>
Ini website pertamaku.
</p>

🏆 Tantangan

Coba ubah kode di atas menjadi: 

<h1>Halo, Namamu!</h1>

<p>
Saya sedang belajar HTML.
</p>
